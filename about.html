About

NRG is a High-end HTML template

Trendy Module Markup

Each page is consisted of independent sections, that are fully interchangeable. You can fit any content you like into the page and it will remain stable no matter what screen a viewer uses a palm-sized handset or huge desktop display.

This will help you to provide a wider range of users with the information about your company, start-up, or whatever good thing you have in mind and you need a web-page in order to achieve it. It looks, cool, modern and trendy as you presumably are!

Usability In Mind

The package contains a well-written, jQuery-based code. It even includes a minified version of the script, which saves some server time, not sensible on a smaller scales but vital for huge server loads, and mobile user's broadband traffic and page load time.

Above all NRG supports touch events when used on a corresponding environment (e.g. touch screen equipped handsets). When using a handset swipe left to open the navigation menu. Native OS tricks in the wild. Pretty crazy right?

Robot Friendly

NRG is a well-structured HTML5 markup. The markup's structure allows search engine bots to scan and process your content in a very proficient way. And in case if the content is relevant to the user lead her at your destination on the world wide web.

NRG is built the same way that I was raised (as a front-end developer) in a strict correspondence with the standards and requirement of World Wide Web Consortium (W3C). Both HTML and CSS are 100% W3C compliant and work very stable.
